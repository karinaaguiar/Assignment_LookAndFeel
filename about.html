About Project

  • How did we achieve responsiveness? The website will automatically rearrange its contents based on the width of the device in which it is being browsed.
    We achieved this by proving widths in Percentages as well as using media queries.
  • Usability Our website has a top navigation bar by which the user can navigate to any of the other pages.
    The user can return to home page just by clicking on the home icon on the navbar.
  • Content Relevance We used Internet sources to get info and images.
    India contents are copied from Wikipedia and 123RF
    Venezuela contents are copied from SpoonUniversity and Touropia
    Vietnam contents are copied from Hotels.com and Touropia
  • Design guidelines followed.
    We have followed below design guidelines for responsiveness.
    • Setting The Viewport.
    • Using the width & max-width Property.
    • Media Queries to accommodate different screen sizes.
