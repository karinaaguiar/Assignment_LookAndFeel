About Project

Responsiveness

The website will automatically rearrange its contents based on the width of the device in which it is being browsed.

We achieved this by proving widths in percentages as well as using media queries.

Usability

Our website has a top navigation bar by which the user can navigate to any of the other pages.

The user can return to home page just by clicking on the home icon on the navbar.

The page is very much user friendly as well.

Design

Below design guidelines for responsiveness.

- Setting The Viewport.

- Using the width & max-width Property.

- Media Queries to accommodate different screen sizes.
